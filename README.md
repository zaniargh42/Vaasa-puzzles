# 🗡️ Helleviu's Footprints (ردپای هله‌ویو)

A hybrid physical-digital puzzle adventure set in Vaasa, Finland, 1867.

## 📖 Overview
**Helleviu's Footprints** is an immersive mystery game where players take on the role of an investigator attempting to solve a mysterious shooting at Sheriff Haglund's home. The game blends traditional physical puzzle-solving with modern digital clues to create a multi-layered detective experience.

## 🎮 Gameplay Mechanics
The game utilizes a "hybrid" approach:
- **Physical Elements**: Players interact with printed newspapers, cardboard stencils, character cards, and hand-drawn maps.
- **Digital Elements**: QR codes are strategically placed to provide audio recordings, visual comparisons, and interactive forms for submitting findings.

## 📍 Journey Map
The investigation spans five key locations in the city of Vaasa:
1. **Vaasa Newspaper Office** 📰 — Decoding archives via stencils.
2. **Sheriff Haglund's House** 🏠 — Logical interrogation and scene analysis.
3. **Crown Warehouse** 📦 — Grid-based navigation and physical rubbing.
4. **Julia Widgren's Studio** 📷 — Visual identification from historic photography.
5. **Old Cemetery** 🪦 — Final deduction and report completion.

## 🎯 Design Philosophy
- **Deductive Reasoning**: Puzzles are designed so that players must synthesize information from previous stages to solve current ones.
- **Observation**: Success depends on paying attention to small details (e.g., a specific color of a coat or the number of metallic sounds in a recording).
- **Atmosphere**: Every element is crafted to evoke the feeling of autumn in 1867 Vaasa.
